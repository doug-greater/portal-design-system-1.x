Anatomy

Shell: solid fill, white text, border-radius: 12px, padding 13px 22px, font: 500 16px Inter. One soft drop shadow — 0 6px 16px rgba(0,0,0,.14).

Color = meaning: success uses --p-success (#00BC57); error uses --p-danger (#E5484D). No other tones — toasts are confirmations and failures only; for anything richer use an Info Banner or Dialog.

Icon: 22px, leading. Success = outline check; error = filled error (FILL 1 reads clearer on a solid fill).

Position: fixed, top-center, top: 18px, above all content. Stack downward with 10px gap when more than one is live.

Behavior: auto-dismiss after ~4s (success) / ~6s (error); pause on hover. Add the ✕ only when the message is persistent or actionable. Slide in with a 180ms ease-out translate (no opacity-only fade), matching panel motion.
