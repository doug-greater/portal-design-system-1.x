AccountTypeIcon — the canonical mark for an Account anywhere it's shown compactly (table leading cell, detail-header avatar, map callouts). White disc + 1.5px #DDE1E6 ring + dark (#1C1C1E) outline glyph; glyph is half the container. ring={false} inside a container that already provides one; pass an explicit icon to override the ACCOUNT_TYPE_ICONS map (default storefront).

AccountTypePill — a neutral category pill (#F3F4F6 / #4A5565, text only) that names the type. Distinct from the colorful category Pill and the dotted lifecycle StatusBadge. Pair with the icon in headers.
