Anatomy & rules

<Highlight text query /> wraps matched substrings in <mark class="gr-hl"> (token --p-highlight / --p-highlight-fg — the only sanctioned yellow). Returns the plain string when nothing matches.

One grammar, mirrored. The frontend lib/searchQuery.js (parseQuery / matchesQuery / matchRanges) and the backend search_utils.py must stay in sync — every list endpoint and the ⌘K global search filter through the same matcher, so a query behaves identically in the URL-bar field, in each list, and in the palette.

Yellow is reserved. Search-highlight is the only place yellow appears in Portal — never for warnings, selection, or emphasis.
