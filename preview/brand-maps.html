Maps

Any time the portal needs a map — routes, account locations, territory planning, coverage analytics — use Leaflet with the CARTO Light All basemap. Never satellite, never colorful tiles. Two map families share this language: Routes (polylines + numbered pins) and the Coverage Map (a D3 hexbin overlay; color = Condition, fill area = Demand velocity).

A Leaflet basemap + a D3 d3-hexbin SVG overlay. Each bin encodes two dimensions: color = average Condition of its stores; fill area = a magnitude (avg Demand velocity). Area — not opacity — carries magnitude: prop = max(0.34, sqrt(avg/max)) inside a fixed outline hex.

  • Leaflet overlay pointer-events. Leaflet sets pointer-events: none on its overlay pane — interactive SVG (.g-hex-outline, .g-cov-pin) must force pointer-events: all !important or hover/click silently never fires. Keep .g-hex-fill / .g-hex-grid / pin labels at none.
  • z-index ladder: overlay cards 500 · hex tooltip 600 · DS Tooltip 4000.
  • Stores mode (ModeBtn) swaps hexbins for .g-cov-pin circles (fill = each store's condition; .is-pending → dashed). Re-render the overlay on mode/filter/spotlight change and on map move/zoom/resize.
  • Dark mode (1.5). The basemap swaps CARTO light_all↔dark_all on the resolved theme — rebuild the tile layer in an effect keyed on resolved (why theme lives in one shared store). Overlay cards/legend/tooltip use tokens and theme for free; only the basemap chrome + raw SVG layers (.g-hex-*, .g-cov-pin) get dark overrides in maps.css. Condition colors use the theme-aware Palette A tokens (brightened in dark — see colors-conditions.html).

  • Add .is-active to the currently selected / in-focus stop — inverted blue fill + halo. Never more than one active pin per map.
  • Add .is-unassigned for stops that haven't been slotted into a route yet. These carry no number; the red ring is the "needs attention" signal.
  • Label font is Inter Bold — don't substitute. Default label color matches the ring (#282838); the active variant flips it to white.

Usage
  • Load Leaflet 1.9.x CSS + JS from unpkg. Include maps.css after leaflet.css.
  • Tile URL: https://{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}{r}.png — the {r} gets @2x on retina automatically.
  • Wrap the map container in <div class="g-map"> to pick up border, radius, elevation, and zoom-control + attribution restyling.
  • When embedding inline on a page (not a floating popover), add .is-inline to drop the shadow and soften the radius.
