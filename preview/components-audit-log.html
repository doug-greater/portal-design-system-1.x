Audit Log, Change Row & Restore

Action accents: Created = --p-primary (flag) · Updated = gold #B7791F (edit) · Restored = teal --p-restore #0D9488 (settings_backup_restore) · Deleted = --p-danger (delete).

Ledger columns are Removed / Added (not Old/New) — correct for scalar edits, multi-select membership, and creations alike. Removed renders struck-through --p-muted; Added in --p-ink. Record Type is plain text — no icon, no pill. Deletions are tombstones (the name persists).

Change Row is shared by the timeline modal and the wizard Review step: scalar (old → new) or membership (+ green / − red chips).

Every non-current version with a snapshot offers Restore This Version (teal) with an inline confirm. The modal opens at the 560px custom width.
