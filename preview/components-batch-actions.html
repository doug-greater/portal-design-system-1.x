Batch Actions (header dropdown)

The default multi-select pattern for operator list / table pages. A Neutral button "Batch Actions" + expand_more sits in the page header, left of the primary "New …" CTA.

Always present, but disabled until selected > 0; then it opens the standard Menu popover (Edit Dates, Edit Qty, Delete — destructive items last, in --p-danger).

Pairs with a header-checkbox select-all (visible page) and the standard table footer (Showing X–Y of Z + RowsSelect + Pagination).

Do not also float a Selection Bar on the same surface — pick one. On tables, that's Batch Actions; reserve the floating Selection Bar for canvas / non-tabular surfaces.
