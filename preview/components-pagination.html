Rules

Footer bar: attaches to the table's bottom (shares its border, --p-surface-alt fill, border-radius: 0 0 8px 8px). Left = result range Showing 1–50 of 3,600 in --p-muted with bold figures; right = rows-per-page select + pager.

Pager: 30px cells, radius 6px. Active page is a solid --p-primary chip; others are ghost (hover --p-surface-tint). Prev/next are bordered icon buttons that disable at the ends. Collapse long ranges with an ellipsis, always keeping first, last, and the current ±1.

Page size: 25 / 50 / 100. Default 50 — matches the Filter Menu's 50-row search cap so the two never disagree.

Paging vs. Load more: use numbered paging for tables operators scan and jump around (Products, Accounts, Users). Use Load More for feed-like or append-only lists where position doesn't matter. Don't use both on one surface.

Numerals use Geist Mono nowhere here — counts read as prose; only in-cell data stays mono.
