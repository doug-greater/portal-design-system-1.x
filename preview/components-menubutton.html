MenuButton = a full-height (36px) Button (any variant) + a trailing expand_more that opens the identical Menu popover used by SplitButton/Kebab. It lines up with sibling buttons in headers and action bars, where a 28px SplitButton would look stunted.

The rule: inside a data-table row → SplitButton or Kebab (28px). Outside tables (header / action bar / toolbar) → MenuButton. Choose variant by intent; default menuAlign="right".
