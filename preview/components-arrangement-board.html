Products at this store that aren't placed in any section appear here. Drag one into a section to place it.

Built on @dnd-kit (core + sortable + utilities) — accessible, keyboard-operable DnD; the only new runtime dependency. Two-column grid: editor minmax(0,1fr) + a sticky 340px Unassigned tray.

Drag rules: a product may live in multiple sections, so the tray emits but never accepts placements (dragging a placement to the tray is forbidden). General Stock sections reject all drops. After any move, self-heal tray membership (unplaced = catalog − placed).

Capacity is soft-required: unset shows amber (not red). The label morphs Capacity → Display Size via gr-label-swap when Display is on. "Set Capacity For All" (section kebab) bulk-applies one value.
