"Palette A" (1.5) supersedes the 1.4 scale. Audited for colour-blindness (deuteranopia/protanopia) and WCAG ≥3:1 against both CARTO basemaps. Diverging, not sequential: the two "bad" ends (OOS orange-red / Heavy-overstock purple) are maximally distinct; Optimal (teal) is the calm middle.

Theme-aware tokens. conditions.js returns var(--cond-…) strings (never hex), so the same scale flips between the light :root and html[data-theme="dark"] values. level is the ordinal severity; averageCondition() rounds a bin's mean level to one color (the map hexbin).

Token names, severity, and helper API are unchanged from 1.4 — only the values swapped. The chip-tint sibling for At Risk is --g-orange-10 / --p-atrisk-strong (Chip atrisk tone).
