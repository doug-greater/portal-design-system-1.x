Anatomy & rules

Structure: centered column — 56px icon badge (--p-surface-tint circle, --p-muted outline icon) · title 600 16px --p-ink · one-line body 14px --p-muted (max ~320px) · action(s).

Pick the action by cause: no-results (search/filter) → a ghost Clear Filters, never a create CTA. Empty set (nothing exists yet) → a primary create CTA. First-run / import → dashed dropzone with primary + a secondary helper (template download). See the full CSV Import flow (validate → preview → commit) in components-csv-import.html.

In-table variant drops its own border/background and lives inside the existing table card, preserving the header row so the page doesn't collapse.

Icon choice mirrors the surface: search_off / filter_alt_off for empty results, the surface's own nav icon (inventory_2, group, route) for empty sets, upload_file for import.

Restraint: no illustrations, no emoji — one outline glyph only. Copy is plain and second-person, matching the system voice.
