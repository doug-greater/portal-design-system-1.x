EmptyArt structure: a soft depth-stack of two faintly-rotated record cards (±7°) behind a front card with three faux content lines (--p-surface-tint) · a floating 60px circular badge (--p-surface, --shadow-float) housing the surface's contextual glyph (28px, --p-text-2) · a small accent dot (8px) + a fainter echo. It's a pure HTML/CSS composition — no SVG asset.

Accent = state: --p-primary blue (info, default) · --p-danger red (error) · --p-success green (success) · --p-warning gold. Auto-selected from the glyph (error|warning|report|block|cancel → red; check|task_alt|done|verified|celebration → green; otherwise blue), overridable via an accent prop. Blue still means state — the accent dot reuses the same rule.

Pick the action by cause (table above): no-results → ghost Clear filters; empty set → primary create CTA; first-run → dashed dropzone + primary + helper; error → neutral Retry; success → no action.

In-table variant renders compact (smaller cards + 50px badge) and drops its own border/background so it sits inside the table card — the header row stays so the page doesn't collapse.

Motion: the badge gently floats (gr-emptyfloat, via .gr-emptybadge); auto-disabled under prefers-reduced-motion.

Restraint: one framed contextual glyph, plain second-person copy, and no emoji. The glyph now lives inside the illustration so in-between states feel as intentional as the buttons.
