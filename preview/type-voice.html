When a link is rendered in a muted / neutral color (gray, as on stat cards or metadata rows), it loses the blue cue — so it must be underlined to read as interactive. Solid 1px underline, text-decoration-color: #C4C9D2, offset 2px. On hover it darkens to --p-ink. Never use a dotted underline.
